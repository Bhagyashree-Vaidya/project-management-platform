# Chewawa Project Management Platform

A  project management platform and task board designed to keep product teams organized. It provides a unified, interactive workspace for tracking sprints, bugs, and daily workplace chaos.

## Use Cases

Built for product teams, developers, and project managers who want a fast, low-friction, and visually engaging alternative to heavy enterprise tracking tools for managing their daily tasks and sprints.

## How to Log In / Access

* **Live URL:** [https://bhagyashree-vaidya.github.io/project-management-platform/](https://bhagyashree-vaidya.github.io/project-management-platform/)
* **Access:** Publicly accessible. No account creation or login required for the live demo.

## Pages & Features

**Interactive Task Board**

* Seamless drag-and-drop functionality allowing users to move tasks between statuses or priority levels without opening additional menus.
* "Sitcom Sprint" interface populated with themed tasks and priorities (P0 to P3).

**Unified Data Views**

* Multiple perspectives on the same data: Prioritized Backlog, Status Board, Current Iteration, Roadmap, Bugs, and My Items.
* A single data model ensures that moving an item in one view instantly updates across all others.

**Command Center & Metrics**

* Quick-access search and a command palette (Command ⌘K) for rapid navigation.
* Custom, humorous metric dashboards tracking "Laugh Velocity," "Cold Opens," "Chaos Level," and "At Risk" items.

## Integrations / Setup

* **Deployment:** Hosted directly via GitHub Pages.
* **Local Setup:** Developers cloning the repository will likely run standard Node.js package commands (e.g., `npm install` and `npm run dev`) to start the local development environment.

## Miscellaneous

* **UI/UX:** Focuses on "fewer clicks," allowing users to open task drawers, make edits, and close them quickly.
* **Mock vs Live Data:** Currently populated with entertaining, sitcom-themed mock data (e.g., tasks assigned to Michael, Leslie, and Chandler) to demonstrate functionality.
