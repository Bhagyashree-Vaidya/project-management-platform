# Chewawa Project Management Platform

Chewawa Project Management Platform is a project management platform hosted with GitHub Pages. It turns the repository into a product-team workspace with a prioritized backlog, status board, current iteration view, roadmap, bug triage, my items, a command palette, and draggable task cards.

## How to access

### Live site

Open the GitHub Pages site:

<https://bhagyashree-vaidya.github.io/project-management-platform/>



## Project files

The app is served directly from the repository root:

- `index.html` contains the page structure.
- `style.css` contains the visual design.
- `script.js` contains the sample tasks and interactions.
